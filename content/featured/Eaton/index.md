At Eaton, a $27B global power management company, I identified a critical gap between how firmware engineers documented APIs and what sales teams, integrators, and external developers actually needed. Existing documentation was inconsistent, incomplete, and creating friction at every handoff.

I led a full overhaul of REST API documentation across 200+ endpoints — validating requests and responses in Postman, establishing a standardized documentation framework, and driving adoption across the product line. The result was a documentation system that reduced integration friction for external developers and became the adopted standard across the team.
